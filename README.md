# API "Телефон-Адрес"
Микросервис для хранения и управления связками "телефон-адрес".  
Позволяет сохранять, получать, обновлять и удалять адреса по номерам телефонов.  
Используется Redis как хранилище данных.

## Возможности API
- Просмотр данных
- Создание новой записи
- Обновление существующей записи
- Удаление записи


## Установка и запуск локально
1. Склонируйте репозиторий:
```
git clone https://github.com/levinadev/phone_address_service.git
cd phone_address_service
```

2. Запустите сервис и Redis в контейнерах:
```
docker compose up --build -d
```

3. После запуска документация Swagger доступна по адресу:
```
http://127.0.0.1:8000/docs
```

4. Для остановки сервисов:
```
docker compose down
```


## Примеры запросов:
1. Создать связку
```
POST /
Content-Type: application/json

{
  "phone": "+7-999-123-45-67",
  "address": "г. Москва, ул. Пушкина, д. 10, кв. 5"
}
```

2. Получить адрес по телефону
```
GET /+7-999-123-45-67`
```

3. Обновить адрес
```
PUT /+7-999-123-45-67
Content-Type: application/json

{
  "address": "г. Москва, ул. Лермонтова, д. 20"
}
```

4. Удалить связку
```
DELETE /+7-999-123-45-67
```
